# AI Trip Planner – LangChain Based

## Short Description
AI Trip Planner is an autonomous agentic system built using **LangChain** and **OpenAI GPT** models. It generates **3–7 day travel itineraries** including:

- Cheapest flights between cities
- Best-rated hotels within budget
- Top places to visit
- Weather forecast for each day
- Estimated total budget
- Day-wise itinerary

The system uses **JSON datasets** for flights, hotels, and places, and **Open-Meteo API** for real-time weather. It is integrated with **tools** for multi-step reasoning and outputs clean, readable trip plans.

---

## Features

- **Autonomous planning** using LangChain agents
- **Multi-tool integration:** flights, hotels, places, weather, budget
- **Strict output formatting** in plain text
- **Budget optimization** & best recommendations
- **CLI & Streamlit interface support**
